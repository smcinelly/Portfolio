Authentic Grace

Fictional women's clothing boutique. Static frontend website. Deployed on Heroku and Github. Using HTML and CSS.

Overview:
Using the Human-Centered Design Thinking process, this was a minimum viable product project assigned at Moms Can & Co's Moms Can Code School training program. Our assignment was to create a front end static clothing boutique with one product page. Our team of two began by developing a buyer persona and then moved to the design of our wireframe and building. We were able to deploy this frontend website on Heroku and GitHub. This project was completed by using HTML, CSS, and jQuery was used for the image gallery.

My role in the project was that of designer, developer, and quality assurance tester.
